Essential Smart Home Hub

Neutron delivers all the core automation features needed for a modern smart home — simple, reliable, and affordable.

Multi-Protocol Support

Built-in support for WiFi, BLE, IR, RF, and basic compatibility for mainstream smart devices.

App-Based Control

Intuitive mobile app for controlling lights, fans, switches, and sensors from anywhere.

Seamless Device Pairing

Quick and user-friendly onboarding of smart devices with minimal setup.

Basic Automation Engine

Create schedules, timer-based triggers, and basic rule-based automations (e.g., motion → light ON).

Local + Cloud Sync

DIY & Installer Friendly

Modular Expansion Ready
